# 0xD837 CameraMode

**Vendor Extension Property**  
Returns or sets the current value of the camera screen.

| Value | Description |
|:--|:--|
| `0x0001` | Shooting screen |
| `0x0002` | Playback screen |
| `0x0003` | Setting screen |
| `0x0004` | Plugin selection screen |

> [!NOTE]
